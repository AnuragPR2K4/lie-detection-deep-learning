# 🧠 Lie Detection & Stress Analysis System using Computer Vision

## 📌 Project Overview

This project presents a real-time **Lie Detection and Stress Analysis System** that analyzes facial behaviour to identify possible signs of stress, discomfort, and deception using computer vision techniques.

The system captures live video through a webcam and processes it using facial landmark detection and behavioural analysis. By tracking key facial regions such as the eyes, nose, and facial muscles, it evaluates patterns including eye movement, eye contact, and subtle facial changes that may indicate stress or emotional tension.

Additionally, the system analyzes facial expressions to better understand the user's emotional state. By combining facial landmark tracking with emotion analysis, it provides continuous real-time behavioural monitoring.

⚠️ This project is intended for research and educational purposes only and is not a scientifically validated lie detection tool.

## ⚙️ System Workflow

1. The webcam captures live facial video input.
2. MediaPipe detects facial landmarks in each frame.
3. Key facial regions (eyes, nose, facial muscles) are tracked.
4. Behavioural patterns such as eye movement and facial changes are analyzed.
5. Emotional cues are evaluated to detect stress indicators.
6. Results and facial landmarks are displayed in real time.

## 🧪 Behavioural Indicators Analyzed

* Eye contact stability
* Blink behaviour

* Subtle expression changes
* Emotional indicators

These cues may correlate with stress, discomfort, or cognitive load.
